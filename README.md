# Bhavya Bhardwari — Portfolio

![Portfolio preview](assets/images/preview.png)

A single-page personal portfolio built from scratch with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools. Designed to bring together my work across writing, technical projects, and community building into one cohesive site.

**Live site:** [https://sapphire-coder-prog.github.io/PortfolioV1/](https://sapphire-coder-prog.github.io/PortfolioV1/)

## About this project

This portfolio showcases:
- **100 Days to the Margins of a Notebook** — a daily public essay series
- **Building** — projects including Lumis (AI academic agent) and Coincide (expense tracking app)
- **BEAN** — a 160+ member engineering student community I co-founded
- **Tech** — my current frontend development skill set and what I'm learning next

It was built as a hands-on project to apply and reinforce frontend fundamentals — semantic HTML, responsive CSS layout with custom properties, and vanilla JavaScript for interactivity.

## Features

- Fully responsive layout, mobile-first design approach
- Custom CSS variable-based theming system, with alternating light/dark sections
- Scroll-triggered fade-in animations using the Intersection Observer API
- Interactive cursor-reactive dot-grid background, built with the Canvas API
- Accessible mobile navigation with keyboard and screen-reader support
- Respects `prefers-reduced-motion` for users with motion sensitivity

## Tech stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, Flexbox, Grid, responsive breakpoints
- **JavaScript (vanilla)** — DOM manipulation, Intersection Observer, Canvas API

No frameworks, no libraries, no build step — just the fundamentals.
